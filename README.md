# Google Sheets × Gemini AI 売上自動分析レポート

売上データを入力してボタンを押すだけで、AIが自動で売上分析レポートを生成します。

## 動作サンプル

※動作サンプル画像は準備中です

## デモ動作イメージ

    【売上データシート】に日次データを入力
             ↓
    メニュー「🤖 AI分析」→「売上レポートを生成」をクリック
             ↓
    Gemini AIがデータを分析
             ↓
    【AIレポートシート】にレポートが自動生成

## 生成されるレポート内容

- 売上トレンドの要約
- 気になる数字・特徴（良い点・悪い点）
- 来週に向けた改善提案（具体的に2点）

## 技術スタック

| 分類 | 技術 |
|------|------|
| スクリプト | Google Apps Script (GAS) |
| データ管理 | Google Sheets |
| AI | Gemini 2.5 Flash Lite API |
| 実行方法 | シート内メニューボタン |

## セットアップ手順

### 1. Google Sheetsの準備

1. 新規スプレッドシートを作成
2. シート名を **「売上データ」** に変更
3. 新しいシートを追加して **「AIレポート」** と名前をつける
4. 「売上データ」シートに以下のヘッダーを入力：

| A列 | B列 | C列 | D列 |
|-----|-----|-----|-----|
| 日付 | 売上（円） | 客数 | 天候 |

### 2. GASの設定

1. スプレッドシートのメニュー「拡張機能」→「Apps Script」を開く
2. コード.gs の内容をエディタに貼り付ける
3. 1行目の YOUR_GEMINI_API_KEY を実際のAPIキーに書き換える
4. 保存してスプレッドシートをリロード

### 3. Gemini APIキーの取得

1. Google AI Studio にアクセス
2. 「Get API Key」→「Create API key」
3. 発行されたキーを コード.gs の1行目に貼り付け

## 使い方

1. 「売上データ」シートに日次データを入力
2. メニュー「🤖 AI分析」→「売上レポートを生成」をクリック
3. 「AIレポート」シートに分析結果が自動表示される

## 活用シーン

- 飲食店・小売店の日次売上管理
- 週次・月次レポートの自動化
- 店長・オーナーへの報告業務の効率化
